### üéÆ The Gameplay Mechanics
Using iterative prompting and collaborative logic design with **Gemini**, I developed a three-stage experience:

1.  **Memory Lane Lane:** A 4x4 reconstructive matrix using our personal photos to "extract" key fragments.
2.  **The Block Forge:** A high-score challenge inspired by *Block Blast*, requiring strategic placement to forge the Masterwork Key.
3.  **The Loot Vault:** A fully functional Hextech Chest that allows the user to **Claim** a common reward or **Reroll** 3 shards into a "Permanent Ultimate" prize.
